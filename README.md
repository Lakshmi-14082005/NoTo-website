

# NoTo - Student Material 

NoTo is a web-based project designed to provide a simple, structured, and user-friendly interface. This repository contains the source code, images, and documentation for the NoTo website.

---

## 📌 Project Overview
- **Name:** NoTo
- **Type:** Web Application
- **Technology Stack:** HTML, CSS

---
💡 Why I Created This:

As a student, I understand how crucial well-organized notes are, especially when preparing for exams and interviews. I created NoTo to provide a simple, accessible space for students to find study materials.

This is my first project, so I kept the design straightforward and avoided complexity to ensure it remains easy to use and maintain. Please note that this project serves as a reference point for organizing study resources.

----
## 🚀 Features
- Intuitive **Home Page** layout for easy navigation.
- Clean and semantic HTML structure.
- Responsive design principles for cross-device compatibility.
- Modular CSS for styling and customization.
